# School Enrollment Analysis Dashboard

Analysis Period: 2022-23 to 2025-26

Live Dashboard: https://alokkmohan.github.io/Enrolment/dashboard/

---

## About

Interactive dashboard for Uttar Pradesh Government School enrollment analysis covering **2,492 schools** across **75 districts** and **822 blocks** over 4 academic years (2022-23 to 2025-26).

## Key Findings

| Metric | Value |
|---|---|
| Total Schools | 2,492 |
| Total Enrollment (2025-26) | 6,07,943 |
| 4-Year Enrollment Growth | +6.9% |
| Growing Schools | 1,414 (56.7%) |
| Declining Schools | 933 (37.4%) |
| HIGH RISK Schools | 404 |
| Model Schools | 382 |
| Red Zone Blocks | 145 |
| Avg Girls Ratio | 58.3% |

## Dashboard Sections

- **Overview** — State KPIs, enrollment trend, risk distribution, school type performance
- **Districts** — 75 district performance table with health tags
- **Blocks** — 844 blocks with zone color-coding (RED / ORANGE / YELLOW / GREEN)
- **Schools** — Search and filter all 2,492 schools by name, district, risk level, type
- **Alerts** — Early warning alerts (1,213 school-level alerts across 10 alert types)
- **Girls** — Girls enrollment growth, district-wise progression analysis
- **Clusters** — 56 same-condition school clusters (systemic patterns)

## Data

Data covers UP Government schools Classes 8–12 across four academic years.
Source data is not included in this repository (private).
